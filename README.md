Разработано полнофункциональное веб-приложение для управления заказами с облачным хранением файлов, которое:

✅ Принимает заказы через веб-форму
✅ Хранит файлы в S3-совместимом облаке (MinIO)
✅ Позволяет скачивать и удалять файлы
✅ Имеет удобный веб-интерфейс
✅ Готово к расширению и масштабированию
✅ Использует современный стек технологий (Java, Spring Boot, Docker)
